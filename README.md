# LLM Applications in Finance

Machine learning project exploring the use of Large Language Models (LLMs) in financial applications.

## Project Overview

This project studies how LLMs can analyze, summarize and extract information from financial texts.

Applications include:
- Earnings calls analysis
- Narrative credit scoring
- Fraud detection
- Financial NLP pipelines

---

## Topics Covered

### Large Language Models
- Transformers
- Instruction tuning
- RLHF
- Distillation
- Mixture of Experts (MoE)

### Advanced LLM Systems
- Retrieval-Augmented Generation (RAG)
- Function Calling
- Semantic search
- Prompt engineering

---

## Financial Applications

### Earnings Calls Analysis
Detection of:
- defensive tone,
- evasive language,
- governance deterioration signals.

### Narrative Credit Scoring
Extraction of qualitative "soft information":
- strategy,
- reputation,
- coherence.

The extracted semantic signals are transformed into machine learning features.

### Fraud Detection
Use of LLMs to:
- detect suspicious patterns,
- identify anomalies,
- reduce false positives.
